a. Перейти в домашнюю директорию:
cd ~


b. Создать новую папку:
mkdir my_new_folder


c. Перейти в новую папку:
cd my_new_folder


d. Создать в новой папке новый файл:
touch my_file.txt


e. Перейти в текстовый редактор (например, nano) и написать текст:
nano my_file.txt 
-> ввод текста


f. Выйти и сохранить этот текст:
В редакторе nano:
Нажмите Ctrl + O для сохранения файла.
Нажмите Enter для подтверждения.
Нажмите Ctrl + X для выхода из редактора.


g. Посмотреть конкретное словосочетание в файле:
grep "ваше_словосочетание" my_file.txt
-> замените "ваше_словосочетание" на нужное слово или фразу.


h. Посмотреть конкретное словосочетание в файле и перенаправить поток в новый файл:
grep "ваше_словосочетание" my_file.txt > new_file.txt

i. Посмотреть, что находится в вашей директории:
ls


j. Как посмотреть, что находится в вашей директории, НО, включая скрытые файлы:
ls -a


k. Посмотреть, где вы находитесь (в плане пути):
pwd


l. Вы в домашней директории -> 
создать 2 папки -> 
перейти в первую любую папку -> 
создать там файл -> 
скопировать и перенести этот файл во вторую папку:

cd ~
mkdir folder1 folder2
cd folder1
touch file_in_folder1.txt
cp file_in_folder1.txt ../folder2/
Теперь файл file_in_folder1.txt будет скопирован в папку folder2.
